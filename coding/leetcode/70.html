    70. 爬楼梯¶

    难度：简单

    题目¶

    假设你正在爬楼梯。需要 n 阶你才能到达楼顶。

    每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？

    注意：给定 n 是一个正整数。

    示例 1：

    输入： 2
输出： 2
解释： 有两种方法可以爬到楼顶。
1.  1 阶 + 1 阶
2.  2 阶

    示例 2：

    输入： 3
输出： 3
解释： 有三种方法可以爬到楼顶。
1.  1 阶 + 1 阶 + 1 阶
2.  1 阶 + 2 阶
3.  2 阶 + 1 阶

    Reference

    题解¶

    基本思路： 使用一个数组存储临时结果，递归计算。

    代码如下：

    int climbStairs_cache(int n, int **cache)
{
    if (n <= 2)
        return n;
    if (*cache == NULL)
        *cache = memset(malloc(sizeof(int) * n), 0, sizeof(int) * n);
    if (!(*cache)[n - 1])
        (*cache)[n - 1] = climbStairs_cache(n - 1, cache) + climbStairs_cache(n - 2, cache);
    return (*cache)[n - 1];
}

int climbStairs(int n){
    int *cache = NULL;
    return climbStairs_cache(n, &cache);
}
