# 应用数理统计

统计学中的两个概念：样本与总体。

* 概率：在总体中进行抽样，得到样本
* 统计：根据样本的统计指标对总体进行推断

## 课程内容

* 有监督模型
    * 线性回归模型、广义线性回归模型
    * Discriminant analysis
    * Cross validation and boostrap
    * Regularization
    * Survival analysis
    * Tree-based methods
* 无监督模型
    * 聚类分析
    * 贝叶斯分析
    * Gibbs sampling and MCMC

## 概率论回顾

1. 条件概率：$P(A|B) = P(AB) / P(B)$
2. 事件独立性：$P(A) = P(A|B)$
3. 贝叶斯定理：$P(A_i|B) = P(A_i) P(B|A_i) / P(B)$
