# 孟加拉

孟加拉是《帝国时代2 决定版：印度王朝》版本中加入的文明，归属于南亚文明

## 文明特色

### 文明加成

1. 所有象受到的附加伤害-25%，招降抵抗等级+1
2. 到达下一时代时，每个现有城镇中心产生2个村民
3. 所有船只每分钟回复15生命值
4. 僧侣类单位护甲+3/盾牌+3

### 独特单位

* 战车（可切换近程/远程战斗模式的骑乘单位）

### 独特科技

* 城堡时代：派（战车和所有象攻击间隔减少16.67%）
* 帝王时代：大乘佛教（村民占用人口-10%）
